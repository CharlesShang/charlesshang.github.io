# charlesshang.github.io

Personal homepage for `https://charlesshang.github.io`.

## Publish

Create a public GitHub repository named `charlesshang.github.io`, then push these files to the `main` branch.

```sh
git init
git add index.html README.md
git commit -m "Create personal homepage"
git branch -M main
git remote add origin git@github.com:charlesshang/charlesshang.github.io.git
git push -u origin main
```

After GitHub Pages finishes publishing, the site will be available at:

```text
https://charlesshang.github.io
```
